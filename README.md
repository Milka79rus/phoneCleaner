# Regular Expressions — Task 2: Phone Cleaner (Optional)
[![Check Code](https://github.com/Milka79rus/phoneCleaner/actions/workflows/run.yml/badge.svg)](https://github.com/Milka79rus/phoneCleaner/actions/workflows/run.yml)
Функция для приведения номеров телефонов разных стран к единому формату: `+7xxxxxxxxxx` или `+<код страны>xxxxxxxxxx`.

## Особенности
- Удаляет пробелы, скобки и тире.
- Автоматически заменяет начальную `8` на `+7` для российских номеров.
- Добавляет `+`, если он отсутствует.
